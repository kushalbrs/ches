# ches
Simple 2-person chess game

1) All the pieces are named with char 
    bR  - Black Rook
    bN  - Black Knight
    bB  - Black Bishop
    bQ  - Black Queen
    bP  - black Pawn
    bK  - black King
    
    wR  - White Rook
    wN  - Obvious
    wB  - Obvio
    wQ  - Of course
    wP  - Really!
    wK  - (I give up) White King
